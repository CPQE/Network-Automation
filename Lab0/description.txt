While studying for the CCNA I read a chapter on network automation and I came upon a short Python snippet of the Netmiko package being used to connect to a couple routers using SSH and configure their syslog and snmp servers. I was inspired to see if I could use GitHub Copilot, which I know from my past experience is more accurate and thorough than ChatGPT, MS Copilot, and Google Gemini at writing code, could help me automatically configure my actual Cisco Catalyst 3560 Switch and 1921 router. While I'm a little scared to actually test it since I don't want to misconfigure or damage my actual hardware, the 2 files of code I've generated, inventory_collector.py and parsers.py, allow connecting to each of these devices, save outputs and configs, compute checksums, and get information about interfaces, VLANs, MAC/ARP tables, routing, Spanning tree information (for the switch) and OSPF configuration (for the router).
